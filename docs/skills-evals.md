# Skill Trigger Evals

Use this file for lightweight trigger examples that help refine skill names, descriptions, and overlap boundaries.

## Purpose

This file helps evaluate whether skills are likely to trigger correctly.

Use it to refine:
- skill names
- skill descriptions
- trigger phrases
- overlap boundaries
